# 魔法工厂 · snownico0722.github.io

一座造好玩又好用小东西的魔法工厂。随便逛,随便拿,记得来玩。

发布在 <https://snownico0722.github.io>。

## 这是什么

不是简历,是个工坊门面。把做出来的项目当"出厂的魔法道具"摆在货架上,炉子里还烧着新的。欢迎大家进来逛。

## 实现

- 明快、暖、会动:齿轮转、炉火烧、火花飘、营业灯闪。
- 纯手写 HTML / CSS / SVG + 一点 JS,无框架,无构建,无图片资源。
- 星标实时从 GitHub API 拉取,失败回落到静态值。

## 文件

- `index.html` — 结构(含内联 SVG 齿轮)
- `style.css` — 全部视觉与动画
- `main.js` — 火花、年份、实时星标(无依赖)

## 本地预览

浏览器直接打开 `index.html`。

## 发布

推送到 `main` 分支即由 GitHub Pages 自动部署。
